Neighborhood-Bulletin-Board:
A Simple Community Posting & Help Request Platform
- Neighborhood Bulletin Board is a community-driven assistance platform where users can request and accept help for various services such as tutoring, home repairs, or volunteering. This creates meaningful connections between those in need and those willing to help.
- Built with Django, GraphQL, MongoDB, and Neo4j, the system efficiently manages user interactions and relationships, ensuring a seamless and engaging experience.
